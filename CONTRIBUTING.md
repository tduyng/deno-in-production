# Contributing Guide 

We welcome your contributions and participation!

## Code of Conduct
 * The company should have publicly shared information that they are using Deno so that anyone can validate it
 * No self promotion allowed
 * Avoid including Deno based products. There is link already added at the bottom of this List to "A curated list of Deno code and resources"("https://github.com/denolib/awesome-deno")

Reminder:
  * This list is not index to all companies out there. The aim of the list is to just give overview.
